# 학습 환경 세팅 가이드
## OpenCV + 딥러닝 제조비전 커리큘럼을 위한 실전 사용법

---

## 1. 도구 3개, 각각 역할이 다르다

```
┌────────────────┐  ┌────────────────┐  ┌────────────────┐
│   코워크 (여기)  │  │  VS Code        │  │  Claude 앱      │
│                  │  │  + Jupyter      │  │  (웹 or 데스크탑) │
│   🎯 튜터       │  │  🔧 코딩        │  │  ❓ 즉석 질문    │
└────────────────┘  └────────────────┘  └────────────────┘
```

### 코워크 = 선생님
- "오늘 뭐 해?" → 오늘 할 실습 안내
- "이거 왜 하는 거야?" → 개념 설명
- "여기까지 했어" → 체크포인트 확인 + 다음 단계 안내
- 커리큘럼 수정, 웹 조사, 파일 생성 같은 무거운 작업도 여기

### VS Code + Jupyter = 작업실
- 코워크에서 받은 실습을 여기서 직접 실행
- .ipynb 노트북으로 코드 + 결과 이미지 + 메모를 한곳에
- 파라미터 바꿔보면서 실험

### Claude 앱 = 사전
- 코딩하다 모르는 게 나오면 여기서 빠르게 검색
- "adaptiveThreshold 파라미터가 뭐야?" 같은 짧은 질문
- 코워크 흐름을 안 깨면서 궁금증 해결

## 2. 하루 학습 루틴 (2~3시간 기준)

### Step 1: 코워크 열기 (10분)
```
나: "다음 거 하자" 또는 "1주차 Day2 시작"
코워크: 오늘 배울 개념 + 실습 내용 안내
```

### Step 2: VS Code에서 코딩 (90~120분)
```
코워크가 알려준 실습을 Jupyter 노트북에서 실행
- AI 자동완성(Copilot)으로 코드 생성
- 파라미터 바꿔보면서 결과 관찰
- 모르는 거 나오면 → Claude 앱에서 빠르게 검색
```

### Step 3: 코워크로 돌아오기 (20~30분)
```
나: "여기까지 했어" 또는 결과 스크린샷 공유
코워크: 체크포인트 확인 + 다음 시간 예고 + 메모리 저장
```

## 3. 핵심 규칙: 코워크는 매번 새 세션

### ❌ 이렇게 하지 마세요
하나의 코워크 대화에서 14주를 쭉 진행
→ 대화가 길어지면 AI가 앞부분 맥락을 잃어버림

### ✅ 이렇게 하세요
```
세션 1: "1주차 Day1 — 이미지 읽기/색공간"     → 끝 → 메모리 저장
세션 2: "1주차 Day2 — 블러/이진화"            → 끝 → 메모리 저장
세션 3: "1주차 Day3 — 모폴로지/윤곽선"        → 끝 → 메모리 저장
세션 4: "1주차 체크포인트 — 복습/확인"         → 끝 → 메모리 저장
...
```

매 세션 끝에 메모리에 진도가 저장되기 때문에,
새 세션에서 "다음 거 하자" 한마디면 바로 이어갑니다.

## 4. 첫날 VS Code 세팅 (한 번만 하면 됨)

### 설치할 것
1. VS Code (https://code.visualstudio.com)
2. Python 3.10 이상 (https://www.python.org)
3. VS Code 확장 프로그램:
   - Python (Microsoft)
   - Jupyter (Microsoft)
   - Pylance
   - GitHub Copilot (무료)

### 터미널에서 패키지 설치
```bash
pip install opencv-python numpy matplotlib ultralytics
```

### 테스트 (VS Code에서 .ipynb 파일 만들어서)
```python
import cv2
import numpy as np
print(f"OpenCV: {cv2.__version__}")
print(f"NumPy: {np.__version__}")
```

이게 되면 준비 완료!
